## Why Compartmental Modeling in Demography?

In demographic analysis, one key objective is to understand how a population evolves over time.
Compartmental modeling is a simple and powerful approach to represent that evolution.
The idea is to split the population into groups (compartments), then model the flows between groups.

This approach is useful to:

- project age structure
- test public policy scenarios
- assess future pressure on healthcare, education, and labor markets

## General Framework

We divide the population into demographic compartments.
In this article, we use three age classes:

- C1: ages 0-14
- C2: ages 15-64
- C3: ages 65+

Notation:

- N1(t), N2(t), N3(t): compartment sizes at year t
- B(t): births
- T12(t): flow from C1 to C2
- T23(t): flow from C2 to C3
- D1(t), D2(t), D3(t): deaths by compartment
- M1(t), M2(t), M3(t): net migration by compartment

The annual discrete model is:

N1(t+1) = N1(t) + B(t) - T12(t) - D1(t) + M1(t)

N2(t+1) = N2(t) + T12(t) - T23(t) - D2(t) + M2(t)

N3(t+1) = N3(t) + T23(t) - D3(t) + M3(t)

## Simple Matrix Form

For fast projections, we can use a transition matrix A:

N(t+1) = A N(t) + m

where:

- N(t) = [N1(t), N2(t), N3(t)]^T
- m is the annual net migration vector

A practical pedagogical parameterization:

- effective fertility: f (births generated by C2)
- transition C1->C2: a12
- transition C2->C3: a23
- survival in C1, C2, C3: s1, s2, s3

Then:

A = [[s1*(1-a12),  f,            0],
     [s1*a12,      s2*(1-a23),   0],
     [0,           s2*a23,       s3]]

## Demographic Example

### Initial Assumptions

Target territory: Guadeloupe-like profile (pedagogical order of magnitude, rounded values).

Initial population (year 0):

- N1(0) = 63,000
- N2(0) = 220,000
- N3(0) = 97,000

Annual parameters:

- s1 = 0.9975
- s2 = 0.994
- s3 = 0.965
- a12 = 1/15 (average duration of 15 years in C1)
- a23 = 1/50 (average duration of 50 years in C2)
- f = 0.0135 (annual births per person in C2)

Annual net migration:

- m = [-350, -900, -120]^T

### Demographic Interpretation

- A fraction of C1 transitions to C2 every year through a12.
- A fraction of C2 transitions to C3 every year through a23.
- Fertility is linked to C2, which approximates the working-age and reproductive-age core.
- Negative net migration captures a population outflow scenario.

### Python Script: 20-Year Projection With Chart

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

# Initial population
N0 = np.array([63000.0, 220000.0, 97000.0])

# Parameters
s1, s2, s3 = 0.9975, 0.994, 0.965
a12, a23 = 1 / 15, 1 / 50
f = 0.0135

A = np.array([
    [s1 * (1 - a12), f, 0.0],
    [s1 * a12, s2 * (1 - a23), 0.0],
    [0.0, s2 * a23, s3],
])

m = np.array([-350.0, -900.0, -120.0])

years = 20
N = np.zeros((years + 1, 3))
N[0] = N0

for t in range(years):
    N[t + 1] = A @ N[t] + m

result = pd.DataFrame(N, columns=["0_14", "15_64", "65_plus"])
result["total"] = result.sum(axis=1)
result["share_65_plus"] = 100 * result["65_plus"] / result["total"]

print(result.head())
print(result.tail())

# Chart: compartment levels + 65+ share
t = np.arange(years + 1)

fig, ax1 = plt.subplots(figsize=(10, 5.6), dpi=140)
ax1.plot(t, result["0_14"], label="Ages 0-14", linewidth=2.2)
ax1.plot(t, result["15_64"], label="Ages 15-64", linewidth=2.2)
ax1.plot(t, result["65_plus"], label="Ages 65+", linewidth=2.2)
ax1.set_xlabel("Projection years")
ax1.set_ylabel("Population size")
ax1.grid(True, alpha=0.25)

ax2 = ax1.twinx()
ax2.plot(t, result["share_65_plus"], label="Share of ages 65+ (%)", linestyle="--", linewidth=2.4)
ax2.set_ylabel("Share of ages 65+ (%)")

lines1, labels1 = ax1.get_legend_handles_labels()
lines2, labels2 = ax2.get_legend_handles_labels()
ax1.legend(lines1 + lines2, labels1 + labels2, loc="upper center", ncol=2, frameon=False)

plt.title("Compartmental Demographic Projection (Guadeloupe-like profile)")
plt.tight_layout()
plt.show()
```

### Expected Results (Interpretation)

With this parameter set, we generally observe:

- a gradual decline in total population
- an increase in the share of ages 65+ (population aging)
- a still-dominant 15-64 group, but with slower long-run dynamics

These outputs are useful to:

- anticipate healthcare and long-term care needs
- estimate school demand through the trajectory of C1
- project potential labor market pressure

## Model Limitations

This first model is intentionally simple:

- fertility is summarized with a single coefficient
- it does not separate sex, narrower age bands, or heterogeneous behavior
- migration is assumed constant
- parameter uncertainty is not yet modeled

In practice, you can improve it with:

- finer age stratification
- multi-scenario analysis (optimistic, central, conservative)
- statistical calibration on historical data
- sensitivity analysis and uncertainty intervals

## Conclusion

Compartmental modeling is a strong starting point for rigorous demographic analysis.
With a few interpretable parameters and a clear flow structure, it already provides decision-useful projections.
